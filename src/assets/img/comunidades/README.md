# Estructura de imagenes para comunidades

Usa esta convencion para guardar fotos y piezas visuales.

- Bucaramanga encuentros:
  - src/assets/img/comunidades/bucaramanga/encuentros/[slug-del-evento]/
- Bucaramanga novedades:
  - src/assets/img/comunidades/bucaramanga/novedades/[slug-de-la-noticia]/
- Portadas de comunidad:
  - src/assets/img/comunidades/bucaramanga/cover.jpg
  - src/assets/img/comunidades/default-cover.svg

Ejemplo:

- src/assets/img/comunidades/bucaramanga/encuentros/meetup-soft-skills-abril-2026/foto-01.jpg
- src/assets/img/comunidades/bucaramanga/encuentros/meetup-soft-skills-abril-2026/foto-02.jpg

Para futuras ciudades:

- src/assets/img/comunidades/bogota/encuentros/[slug]/
- src/assets/img/comunidades/medellin/encuentros/[slug]/

Sugerencia de uso en codigo:

- Usa la portada de Bucaramanga cuando exista una imagen propia.
- Para ciudades sin portada propia, usa src/assets/img/comunidades/default-cover.svg.
